Chapter Twenty-One: 2D Arrays

Activity: Gold Rush

The Gold Rush Mining Company has a Surveyor robot that can check out a plot of land to see how many ounces of gold are present. You are going to complete a program that uses this robot to evaluate several sections within a plot and find the best section with the most ounces of gold. You'll use your knowledge of ArrayLists and 2D arrays to complete this activity!

Project Requirements

Your program should analyze one plot of land. Each plot is identified by an integer number that is entered by the user.

What plot number would you like to survey? 1

Each plot of land is broken into sections. Each section can be visualized as a square, 2D grid. Every cell in the grid is an integer that holds the number of ounces of gold the Surveyor robot will find at that particular spot.

The survey results for a particular plot of land can be stored in an ArrayList. Each element of the ArrayList is one 2D grid of integers. Every integer will be between 0 and 9, representing the number of ounces of gold the Surveyor robot found at that spot.

Program Data Structures

2D arrays and ArrayLists are used throughout this program. We track information as follows:

Data Structure Description
int[][] A 2D array of integers represents one grid in the overall plot. The grid is square with Surveyor.GRID_SIZE cells in each direction. Each cell contains an integer from 0 to 9 representing ounces of gold.
ArrayList<int[][]> A plot contains many grids as defined by Surveyor.NUM_GRIDS. Information on each grid is stored in a 2D array as described above. The 2D arrays for each grid are tracked in an ArrayList.
ArrayList<Integer> Each grid contains a total amount of gold, found by adding together all of the gold in each of the 2D cells. The total gold found in each grid is tracked by an ArrayList of Integers.

import java.util.*;

public class Surveyor 
{
	// each grid will be a square with this many cells
	public final int GRID_SIZE = 3;
	public final int NUM_GRIDS = 3;
	
	// random number generator used to create survey samples
	private Random rand;
	
	// The constructor will use the plot number to initialize the random number generator.
	// No student changes are needed.
	public Surveyor(int plot)
	{
		rand = new Random(plot);
	}
	
	// The surveyPlot() method will produce the survey results for the current plot of land.
	// No student changes are needed.
	public ArrayList<int[][]> surveyPlot()
	{
		// create a new ArrayList that holds a 2D integer array in each element
		ArrayList<int[][]> survey = new ArrayList<int[][]>();
		
		// loop NUM_GRIDS times to create each grid
		for (int i=0; i<NUM_GRIDS; i++)
		{
			// create a new 2D integer array with GRID_SIZE rows and columns
			int[][] grid = surveyGrid();

			// add the grid to the survey ArrayList
			survey.add(grid);
		}
		
		// return the survey results to the calling program
		return survey;
	}
	
	// This method will create one 2D grid filled with ounces of gold in each cell.
	// THE STUDENT WILL COMPLETE THIS METHOD
	private int[][] surveyGrid()
	{

	}
	
	// This method will return the number of ounce of gold found at the current location.
	// No student changes are needed.
	private int sample()
	{
		// return a random number between 0 and 9, inclusive
		return rand.nextInt(10);
	}
}

private int[][] surveyGrid()
Copy

This method is responsible for creating a 2D array representing one grid. Each integer value in the array represents several ounces of gold, which can be generated by calling the sample() method for each cell.

  1. Declare a local variable of type int[][] and initialize it with a new object.
  2. Set up nested for() loops to iterate over all of the rows and columns in the grid. You will have GRID_SIZE rows and columns in each grid. The outer loop should iterate over rows and the inner loop over columns.
    • Inside the innermost loop, call the sample() method, which will return an int. Store this int in the grid at the current row and column. Be sure to use row-major order, meaning the first index into the 2D array is the row: [row][col].
  3. When the 2D array is fully initialized, return a reference to that array from the method.
